# BallballRun

### 休闲
仿造球球大作战，玩家通过控制自己躲避大球吃掉地图中随机生成的各类形状小球或者比自身重量更小的玩家进行游戏。
地图场景为3倍手机尺寸大小，视野随玩家移动而移动，保持玩家小球处于屏幕中心视野最大的状态。
### 安卓游戏
使用sdk 19
```xml
<uses-sdk
    android:minSdkVersion="19"
    android:targetSdkVersion="19" />
```
```properties
target=android-19
```
